About

Welcome to the unofficial Oregon State Beavers Minecraft Server page! This is a survival server for the latest Bedrock and Java editions. Crossplay is enabled so players on Bedrock can interact with players on Java. Theres not to much available besides the traditional Survival experience right now, but minigames and a hub are planned to be added in at a later time. We hope you enjoy!
Go Beavs!

How To Join

Minecraft Java Edition: Run Minecraft 1.19 or above -> Navigate to the "Multiplayer" tab -> Select "Add Server" -> Enter a Server Name and play.osuminecraft.us for the Server Address then select "Done" -> Refresh the page and you're ready to play!

Minecraft Bedrock Edition: Open Minecraft -> Select "Play" -> Navigate to the "Servers" tab -> Scroll to the bottom of the "Featured Servers" column -> Select "Add Server" -> Enter a Server Name and play.osuminecraft.us for the Server Address, make sure the port is set to 19132 -> Select "Save" -> The server will appear under "Additional Servers" at the bottom -> You're ready to play!
